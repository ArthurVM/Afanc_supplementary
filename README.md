# Afanc supplementary materials

Supplementary data, manifests, accession lists, benchmark materials, and
worked modelling examples used with Afanc.

## Contents

| path | description |
|---|---|
| `afanc_database_manifest.xlsx` | Manifest for the Afanc database resources. |
| `supplementary_materials_benchmark.ods` | Benchmark supplementary spreadsheet. |
| `afanc_modelling_example/` | Minimal *Plasmodium falciparum* geolineage modelling example using a Pf7 subset. |

## Modelling example

The `afanc_modelling_example/` directory contains a small, runnable example for
building sparse Afanc-style geolineage models from an Ardal allele matrix and
sample metadata. See
[`afanc_modelling_example/README.md`](afanc_modelling_example/README.md) for the
input metadata description, cohort summary, sample-location map, and workflow
notes.
